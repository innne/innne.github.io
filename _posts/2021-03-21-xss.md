# WebGoat - XSS (1) 

#### XSS에 대해 알아보기

<br/>

> XSS(Cross-Site Scripting)이란 웹 브라우저에서 악의적인 사용자가 스크립트를 삽입하여 의도치 않은 행동을 수행시키거나 민감한 정보를 탈취하는 취약점 공격 방법이다.

<br/>

간단한 예시를 따라해보자.

브라우저의 주소 표시줄에 다음과 같이 입력해본다.

```shell
javascript:alert("XSS Test");
javascript.alert("document.cookie");
```
